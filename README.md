## 🎧 Spotify Clone

Projeto desenvolvido com foco em praticar fundamentos de desenvolvimento web, recriando a interface de um player estilo Spotify utilizando **HTML, CSS e JavaScript**.
link do site: https://projeto-spotify-frontend-eta.vercel.app/

## 🚀 Sobre o Projeto

Este projeto consiste em uma réplica visual e funcional básica do Spotify, com o objetivo de treinar:

- Estruturação de páginas com HTML
- Estilização moderna com CSS
- Manipulação de elementos e eventos com JavaScript

A aplicação simula um player de música, com funcionalidades simples de interação.

## 🛠️ Tecnologias Utilizadas

- **HTML5** → Estrutura da aplicação  
- **CSS3** → Estilização e layout (Flexbox / Grid)  
- **JavaScript (Vanilla JS)** → Interatividade  

---

## 🎯 Funcionalidades

- ▶️ Play e Pause de música  
- ⏭️ Avançar e voltar faixas  
- 🎚️ Barra de progresso interativa  
- 🔊 Controle de volume  
- 📃 Lista de músicas  

---

## 💡 Objetivo do Projeto

Este projeto foi desenvolvido com o intuito de:

- Praticar conceitos básicos de front-end  
- Melhorar a organização de código  
- Trabalhar com manipulação do DOM  
- Simular um projeto real de interface
